# 📖 Mode d'emploi - Boîte à idées citoyenne

## 🎯 Comment utiliser cette application

### 💡 Proposer une idée
1. Cliquez sur le bouton **"Proposer une idée"** (rouge)
2. Rédigez votre idée dans le champ de texte
3. Ajoutez des **#tags** (ex: #logement, #mobilité, #environnement)
4. Email optionnel (avec consentement RGPD requis)
5. Cliquez sur **"Ajouter"**

### 🗳️ Voter pour une idée
1. Sélectionnez votre type de votant : **Résident, Alentour, Étudiant ou Visiteur**
2. Cliquez sur **"👍 Voter"** (bleu) sous l'idée de votre choix
3. Vous ne pouvez voter qu'une seule fois par idée

### 🏷️ Filtrer les idées
- Cliquez sur les **tags colorés** pour filtrer les idées
- Cliquez sur **"Tous"** pour réinitialiser les filtres
- Le graphique s'ajuste automatiquement

### 💬 Commenter une idée
1. Cliquez sur **"Afficher les commentaires"** sous une idée
2. Entrez votre **nom, pseudo ou email**
3. Rédigez votre commentaire (support **Markdown** : `**gras**`, `*italique*`, ``code``)
4. Cliquez sur **"Ajouter commentaire"**

### ✏️ Modifier/supprimer un commentaire
- Seul **l'auteur original** peut modifier/supprimer ses commentaires
- L'authentification se fait via votre navigateur (localStorage)
- Les boutons **"Modifier"** et **"Supprimer"** apparaissent automatiquement

### 📊 Comprendre le graphique
- Les **barres colorées** représentent les priorités par catégorie
- Les couleurs correspondent aux tags principaux
- La hauteur indique le nombre total de votes

### 🔗 Partager l'application
- Utilisez le bouton **"Copier le lien"** en bas de page
- Le lien permet de partager l'application avec les mêmes idées et votes

## 🎨 Formatage Markdown disponible

- `**Texte en gras**` → **Texte en gras**
- `*Texte en italique*` → *Texte en italique*  
- ``Code inline`` → `Code inline`
- Retours à la ligne respectés
